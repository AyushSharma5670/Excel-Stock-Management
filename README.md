# Excel Stock Management Dashboard

This project is an **Excel-based inventory management system** for tracking products, stock levels, and reorder alerts.

## Features
- Track stock levels using **formulas** (Current Stock = Stock In − Stock Out)  
- Automatically flag low stock using **IF formula**  
- Analyze stock by **category and top-selling products** using **Pivot Tables**  
- Dashboard with **charts** for visual analysis  
- Conditional formatting highlights low stock items

## File
- `Stock_Management.xlsx` → Contains dataset, formulas, pivot tables, and dashboard

## How to Use
1. Open `Stock_Management.xlsx` in Microsoft Excel.  
2. Update stock data in the dataset sheet.  
3. Pivot Tables and charts will automatically update.  
4. Low stock items are highlighted in **red**.
